# Project Development Phase

Overview:


The Project Development Phase focuses on implementing the planned solution for the Heart Disease Analysis project. In this phase, the dataset is collected and stored in a structured database. SQL operations are performed to clean, filter, and prepare the data for analysis. Tableau is then used to create interactive visualizations and dashboards that highlight key risk factors related to heart disease. Finally, the dashboards are integrated into a web interface using Flask, allowing users to interact with the visualizations and gain meaningful insights from the data.

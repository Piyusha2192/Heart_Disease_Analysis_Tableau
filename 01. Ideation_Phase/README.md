# Brainstorming – Idea Generation – Prioritization


The team discussed multiple ideas related to healthcare data analysis and visualization. After evaluating feasibility and impact, the idea of Heart Disease Analysis using Tableau dashboards was selected to identify key health risk factors through interactive visualizations.


# Define Problem Statements


Heart disease is one of the leading causes of death worldwide. The project aims to analyze heart disease data using Tableau visualizations to identify patterns and risk factors that contribute to cardiovascular diseases.


# Empathy Map Canvas


The empathy map helps understand the needs of users such as doctors, policymakers, and individuals. It ensures the system provides clear insights that help users understand heart disease risks and make better health decisions.
